Current Alerts

  • 18/09/2006 - Still rise in connections per day by 300% which indicates it got worse so.

Solutions

  • Switch off the network only to people on the internal LAN Network or entirely for a period of time to fix the security system
  • Improve the DNS Servers strength and bandwidth so DDos Attacks don't impact as much
  • Deploy anti-DDoS hardware and software modules
  • Configure network against DDoS Attacks
